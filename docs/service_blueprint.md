# 서비스 블루프린트

## 서비스명

안산 세이프코스트 AI

## 서비스 목적

안산 연안 공간에서 발생할 수 있는 안전사고를 사후 대응이 아니라 사전 예방 관점에서 줄인다. 시스템은 안산시가 “언제, 어디에, 어떤 예방 개입을 해야 하는지” 판단하도록 돕는다.

## 서비스 구성

```text
[공공·민간 데이터 수집]
        ↓
[시공간 위험도 예측 엔진]
        ↓
[익명 현장 상황 인식]
        ↓
[예방 개입 추천]
        ↓
[안산시 행정 대시보드]
        ↓
[문자/전광판/관광안내/관계기관 공유]
        ↓
[개입 효과 분석 및 정책 리포트]
```

## 핵심 모듈

### 1. 위험도 예측 모듈

입력:

- 조위/물때
- 기상 예보 및 실황
- 일몰/야간 여부
- 요일/공휴일/성수기
- 유동인구/방문객 흐름
- 사고이력 및 위험구역 데이터
- 교통/주차 혼잡 후보 데이터

출력:

- 장소별 위험도 등급
- 시간대별 위험도 변화
- 위험 상승 원인 설명

예시 출력:

> 오늘 17:00~20:00 방아머리 인근 위험도 높음. 원인: 주말 방문객 증가 + 일몰 전후 체류 증가 + 조위 변화 + 강풍 예보.

### 2. 익명 상황인식 모듈

목표는 사람을 알아보는 것이 아니라, 위험한 상황이 만들어지는 패턴을 감지하는 것이다.

감지 후보:

- 금지/주의 구역 방향 접근 증가
- 방파제 끝단 체류 증가
- 야간 시간대 해안가 체류 증가
- 갯벌 방향 이동 후 복귀 지연
- 강풍·고조위 시간대 해안 근접 체류
- 가족 단위·어린이 동반 추정 군집의 위험구역 접근

금지해야 할 기능:

- 얼굴인식
- 차량번호 인식
- 개인별 이동경로 추적
- 개인 위험점수 부여
- 특정 개인 대상 제재

### 3. 예방 개입 추천 모듈

안산시가 할 수 있는 조치를 추천한다.

개입 유형:

| 유형 | 예시 |
|---|---|
| 정보 안내 | 재난문자, 카카오채널, 관광안내 페이지, 주차장 안내판 |
| 현장 안내 | 전광판 문구 변경, 위험시간 안내 방송, 임시 안내 배너 |
| 인력 운영 | 안전요원 배치 권고, 순찰 협조 요청, 근무 시간대 조정 |
| 시설 개선 | 안내판 위치 개선, 야간 조명 보강, 안전펜스 우선 설치 후보 도출 |
| 관계기관 공유 | 해경·소방·경찰에 위험 예측 리포트 공유 |
| 정책 개선 | 반복 위험구간에 대한 예산·시설 투자 근거 제공 |

### 4. 행정 대시보드

대시보드는 시민 앱보다 중요하다. 심사위원에게 보여줄 최종 MVP의 중심이다.

필수 화면:

1. **오늘의 연안 위험지도**
   - 지도 위 위험도 색상 표시
   - 시간 슬라이더
   - 위험 상승 요인 설명

2. **위험상황 알림 패널**
   - “주의 필요”, “개입 권고”, “관계기관 공유 권고” 구분
   - 원인 데이터 표시

3. **개입 추천 카드**
   - 추천 조치
   - 예상 효과
   - 담당 부서
   - 실행 난이도
   - 권한 주체 구분

4. **효과 분석 리포트**
   - 개입 전후 혼잡 변화
   - 위험구역 체류 감소 여부
   - 신고/민원 변화
   - 다음 개선안

## 사용자 시나리오

### 시나리오 A: 주말 대부도 관광객 집중

1. 토요일 오전, 시스템이 오후 4~8시 특정 구역 위험도 상승을 예측한다.
2. 원인은 조위 변화, 일몰, 주차장 혼잡, 유동인구 증가로 표시된다.
3. 대시보드는 안산시에 전광판 문구 변경과 관광안내 채널 사전 알림을 추천한다.
4. 필요 시 관계기관에 “고위험 시간대 예측 리포트”를 공유한다.
5. 다음 날 시스템은 알림 전후 체류 패턴 변화를 리포트로 제공한다.

### 시나리오 B: 갯벌 방향 이동 후 복귀 지연

1. 특정 시간대 갯벌 방향 이동량이 증가한다.
2. 조위 상승 시간이 가까워지면서 위험도가 급상승한다.
3. 시스템은 안내방송·전광판·관광안내 알림을 추천한다.
4. 안산시는 현장 관리자 또는 관계기관에 공유한다.
5. 시스템은 위험시간 이후 잔류 추정 군집 감소 여부를 확인한다.

### 시나리오 C: 반복 위험구간 시설 개선

1. 한 달 동안 특정 지점에서 위험도 상승과 현장 체류 패턴이 반복된다.
2. 시스템은 “안내판 위치 개선”, “야간 조명”, “동선 유도 시설” 후보로 제시한다.
3. 안산시는 예산 편성·시설 개선 우선순위 근거로 활용한다.

## 서비스가 앱이 아닌 이유

시민 앱은 선택 사항이다. 핵심은 안산시가 위험을 먼저 보고 움직이는 구조다.

- 시민이 앱을 깔지 않아도 전광판·문자·관광안내·주차장 안내판으로 전달 가능
- 담당자는 대시보드로 위험 원인과 조치를 확인 가능
- 반복 데이터를 정책 개선에 활용 가능

## MVP 범위

### 1단계 MVP

- 공개 공공데이터 기반 위험지도
- 장소별/시간대별 위험도 예측
- 개입 추천 카드
- 리포트 자동 생성
- 가상 또는 샘플 데이터 기반 데모

### 2단계 확장

- 현장 CCTV/센서의 익명 카운팅 연동
- 전광판/안내방송 API 또는 운영 프로세스 연동
- 관계기관 공유 리포트 자동화

### 3단계 고도화

- 개입 효과 학습
- 시범구간 확대
- 다른 연안 도시/하천/호수 관광지로 확산

# 발표자료 목차 v1

## 발표 기준

- 발표 시간: 15분 이내
- 권장 장수: 14장
- 발표 전략: 문제 제기보다 `데이터와 AI가 어떻게 행정 조치로 연결되는지`를 선명하게 보여준다.

## 슬라이드 구성

### 1. 제목

**안산 세이프코스트 AI**  
익명 상황인식 기반 연안 고위험상황 선제예방 시스템

핵심 메시지:

```text
안산시는 사고가 난 뒤 알게 되는 것이 아니라, 위험한 시간이 오기 전에 미리 볼 수 있습니다.
```

### 2. 문제 제기

내용:

- 연안 안전 위험은 단일 요인이 아니라 조위, 기상, 방문객, 공간 구조가 겹칠 때 커진다.
- 기존 대응은 사고 후 신고·출동 중심으로 보이기 쉽다.
- 필요한 것은 사고 전 위험 조합을 포착하는 예방 행정 체계다.

시각자료:

- `사고 후 대응`과 `사고 전 예방` 비교 다이어그램

### 3. 왜 안산시인가

내용:

- 안산은 연안·도서·관광·도시 기능이 결합된 지자체다.
- 대부도 등 연안 활동 공간에서 관광, 낚시, 갯벌 체험, 해안 산책이 복합적으로 발생한다.
- 특정 장소의 수치와 사례는 공식 자료로 검증 후 반영한다.

시각자료:

- 안산 연안 후보 구역 지도

### 4. 제안 개요

내용:

```text
조위·기상·유동인구·사고이력·공간정보·익명 현장 신호를 결합해
장소-시간별 위험도를 예측하고,
안산시가 수행 가능한 예방 조치를 추천하는 행정 의사결정 시스템
```

시각자료:

- 서비스 한 줄 개념도

### 5. 전체 파이프라인

내용:

```text
데이터 수집
→ 장소-시간 표준화
→ 위험 피처 생성
→ AI 위험도 예측
→ 위험 원인 설명
→ 예방 개입 추천
→ 담당자 실행
→ 효과 피드백
```

강조:

- 단순 앱/챗봇이 아니라 행정 판단 엔진이다.

### 6. 활용 공공데이터

내용:

| 데이터 | 활용 |
|---|---|
| 조위 | 만조·간조, 조위 상승 속도 |
| 기상 | 풍속, 강수, 특보 |
| 연안사고 | 사고 유형·장소·원인 패턴 |
| 유동인구 | 방문객 집중 시간대 |
| 공간정보 | 갯벌, 방파제, 주차장, 안전시설 |

강조:

- 위험은 데이터 하나로 판단할 수 없고 결합이 필요하다.

### 7. AI 모델 구조

내용:

- 시공간 위험도 예측
- 이상 혼잡 탐지
- 익명 위험행동 패턴 인식
- 예방 개입 추천
- 개입 효과 피드백

시각자료:

```text
위험 피처 테이블
→ 위험도 예측 모델
→ 원인 설명
→ 개입 추천 모델
→ 효과 학습
```

### 8. MVP 화면 1: 오늘의 위험지도

내용:

- 지도 위 위험지점 마커
- 시간 슬라이더
- 위험도 색상
- 우선 확인 구간 목록

발표 멘트:

```text
담당자는 오늘 어느 시간, 어느 구역이 위험한지 먼저 확인합니다.
```

### 9. MVP 화면 2: 위험도 상세와 원인 설명

내용:

- 위험도 82점 / 높음
- 만조까지 47분
- 유동인구 평소 대비 165%
- 풍속 상승
- 일몰 시간대 진입

강조:

- AI 판단을 설명 가능한 방식으로 제공한다.

### 10. MVP 화면 3: 예방 개입 추천

내용:

- 전광판 문구 변경
- 관광안내 채널 사전 알림
- 관계기관 공유 리포트
- 시설 개선 후보 등록

강조:

- 위험을 보여주는 데서 끝나지 않고 안산시가 할 일을 추천한다.

### 11. 관계기관 공유와 권한 경계

내용:

- 해경의 구조·통제 권한을 대체하지 않는다.
- 안산시는 안내, 시설, 관광, 관계기관 공유 중심으로 역할을 수행한다.
- 출입통제 자동화, 구조 지휘, 개인 제재는 제외한다.

발표 멘트:

```text
이 시스템은 통제 시스템이 아니라 예방 행정 지원 시스템입니다.
```

### 12. 개인정보 보호 설계

내용:

- 얼굴인식 없음
- 개인추적 없음
- 개인별 위험점수 없음
- 원본 영상 미저장 또는 최소화
- 구역 단위 인원 수·방향·체류시간만 활용

시각자료:

- `개인 식별 데이터 제외`와 `비식별 집계 신호 사용` 비교

### 13. 기대효과와 검증 지표

내용:

| 효과 | 검증 지표 |
|---|---|
| 위험 노출 감소 | 위험시간대 체류 추정 인원 변화 |
| 행정 효율 향상 | 조치 소요시간, 우선순위 정확도 |
| 협업 강화 | 관계기관 사전 공유 건수 |
| 정책 개선 | 반복 위험구간 시설 개선 후보 |

강조:

- 사고 건수만으로 단기 평가하지 않고 선행지표를 함께 본다.

### 14. 추진계획과 확산 가능성

내용:

1. 공개데이터 기반 MVP
2. 시범구간 비식별 현장 신호 연동
3. 효과 검증 후 안산 내 확대
4. 다른 연안 도시, 하천, 호수, 축제장으로 확산

마무리 메시지:

```text
안산 세이프코스트 AI는 공공데이터와 AI를 사고 후 대응이 아니라 사고 전 예방 행정으로 연결하는 제안입니다.
```

| 구간 | 슬라이드 | 시간 |

| 서비스와 데이터·AI | 4~7 | 4분 |
| MVP 시연 | 8~10 | 4분 |

## 발표 시 피해야 할 표현

- AI가 시민을 감시한다
- 고위험자를 추적한다
- 안산시가 출입통제를 자동화한다
- AI가 구조를 지휘한다
- 사고를 완전히 막는다

## 발표 시 권장 표현

- 고위험상황을 조기에 식별한다
- 익명·집계 기반으로 현장 패턴을 본다
- 예방 행정 판단을 지원한다
- 관계기관 협업 근거를 제공한다
- 위험 노출을 줄인다

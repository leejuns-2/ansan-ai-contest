# 제출용 아이디어 기획서 v1

## 1. 아이디어명

**안산 세이프코스트 AI**  
익명 상황인식 기반 연안 고위험상황 선제예방 시스템

## 2. 제안 배경

안산시는 대부도, 항구, 갯벌, 해안 산책로 등 연안 활동 공간을 보유한 도시다. 이 공간에서는 관광, 낚시, 갯벌 체험, 해안 산책, 주차·이동이 한꺼번에 발생한다. 연안 안전 위험은 단순히 날씨가 나쁘거나 사람이 많다는 이유만으로 발생하지 않는다. 조위 변화, 기상 악화, 일몰, 유동인구 집중, 과거 사고 유형, 공간 구조, 현장 체류 패턴이 겹칠 때 위험이 급격히 높아진다.

기존 안전 대응은 사고 발생 후 신고·출동·구조 중심으로 인식되기 쉽다. 그러나 지방자치단체가 공공데이터와 AI를 활용하면 사고가 발생하기 전 위험이 높아지는 시간·장소·상황을 예측하고, 시민 안내, 현장 안내 강화, 시설 개선, 관계기관 공유 등 예방 행정 조치를 선제적으로 수행할 수 있다.

본 제안은 안산시가 연안 사고를 직접 통제하거나 구조 지휘하는 시스템이 아니다. 안산시가 수행 가능한 예방 행정 수단을 데이터 기반으로 더 정확하고 빠르게 선택하도록 지원하는 AI 의사결정 시스템이다.

## 3. 문제 정의

현재 연안 안전관리에는 다음 한계가 있다.

1. 위험 요인이 조위, 기상, 방문객, 공간 특성 등 여러 데이터에 흩어져 있어 종합 판단이 어렵다.
2. 사고 발생 후 대응은 가능하지만, 사고 전 위험 조합을 조기에 포착하기 어렵다.
3. 안전 인력과 안내시설은 제한되어 있어 어느 시간과 장소에 우선 배치해야 하는지 판단 근거가 필요하다.
4. 시민 안내는 정적인 안전수칙 중심이라 시간·장소별 위험 변화에 맞춘 동적 안내가 부족하다.
5. 현장 상황을 파악하더라도 개인정보 침해 없이 익명·집계 방식으로 처리하는 설계가 필요하다.

따라서 안산시에 필요한 것은 단순 알림 앱이 아니라, 공공데이터와 AI를 결합해 고위험 시간·장소·상황을 예측하고 예방 조치를 추천하는 행정 의사결정 시스템이다.

## 4. 서비스 개요

안산 세이프코스트 AI는 조위·기상·연안사고 이력·유동인구·공간정보·익명 현장 신호를 장소-시간 단위로 결합해 안산 연안의 위험도를 예측하고, 안산시가 수행 가능한 예방 조치를 추천한다.

```text
공공·행정·현장 데이터 수집
→ 장소-시간 표준화
→ 위험 피처 생성
→ AI 위험도 예측 및 이상상황 탐지
→ 위험 원인 설명
→ 예방 개입 추천
→ 담당자 검토 및 실행
→ 관계기관 공유·시민 안내·시설 개선
→ 개입 효과 분석 및 정책 피드백
```

핵심 기능은 네 가지다.

| 기능 | 설명 |
|---|---|
| 연안 위험도 예측 | 장소·시간별 위험도 점수와 위험 등급을 산출한다. |
| 위험 원인 설명 | 만조 임박, 유동인구 증가, 풍속 상승 등 위험 상승 원인을 표시한다. |
| 예방 개입 추천 | 전광판, 안내방송, 관광안내 채널, 시설 개선, 관계기관 공유 등 조치를 추천한다. |
| 효과 피드백 | 개입 전후 체류, 접근량, 신고·민원, 공유 건수 변화를 분석한다. |

## 5. 활용 공공데이터

| 데이터 | 제공기관 | 활용 내용 |
|---|---|---|
| 연안사고 이력 | 해양경찰청 | 사고 유형, 장소 유형, 시간대, 원인 패턴 분석 |
| 연안안전 예측분석모델 | 해양경찰청 | 기존 격자형 위험도와 안산 연안 위험 후보 비교 |
| 조위 실측·예측 | 국립해양조사원 | 만조·간조, 조위 상승 속도, 만조까지 남은 시간 산출 |
| 기상 예보·실황 | 기상청 | 풍속, 강수, 기온, 기상특보 등 위험 요인 반영 |
| 유동인구/생활인구 | 경기데이터드림 등 | 관광객 집중 시간대, 평소 대비 혼잡도 추정 |
| 공간정보 | 안산시·공간정보 데이터 | 갯벌, 방파제, 주차장, 관광지, 안전시설과의 관계 분석 |
| 공영주차장·관광시설 | 안산도시공사·안산시 | 방문 집중 보조지표, 안내 채널 설계 |
| 현장 비식별 신호 | 시범구간 내부 연계 후보 | 사람 수, 이동 방향, 체류시간 등 익명 상황인식 |

1단계 MVP는 공개 공공데이터 중심으로 구현하고, 현장 CCTV/IoT 신호는 시범구간에서 비식별 집계 방식으로 단계적 연동한다.

## 6. AI 분석·활용 방법

### 6.1 장소-시간 위험 피처 생성

모든 데이터를 위험지점과 시간대 기준으로 통합한다.

```text
datetime
location_id
tide_level
minutes_to_high_tide
tide_change_speed
wind_speed
rainfall
visitor_index
visitor_change_rate
accident_history_score
spatial_risk_score
risk_score
risk_level
```

### 6.2 시공간 위험도 예측

조위, 기상, 유동인구, 사고이력, 공간 특성을 결합해 장소·시간별 위험도를 예측한다. 초기에는 설명 가능한 규칙 기반 모델과 로지스틱 회귀, 랜덤포레스트, Gradient Boosting 등 해석 가능한 모델을 활용한다. 데이터가 축적되면 시계열 모델이나 시공간 그래프 모델로 고도화한다.

출력 예시:

```text
위험도 82점 / 높음
주요 원인:
1. 만조까지 47분
2. 유동인구 평소 대비 165%
3. 풍속 상승
4. 일몰 시간대 진입
5. 과거 유사 조건 사고 이력
```

### 6.3 이상 혼잡 탐지

평소 동일 시간대와 비교해 방문객 또는 체류 인원이 급증하는 구간을 탐지한다. 초기에는 이동평균 대비 편차와 전주/전월 동일 시간대 대비 증가율을 사용하고, 고도화 단계에서는 Isolation Forest 등 이상탐지 모델을 적용한다.

### 6.4 익명 상황인식

시범구간에서 현장 영상 또는 센서를 활용할 경우, 개인을 식별하지 않고 구역별 인원 수, 이동 방향, 체류 시간 등 비식별 집계 신호만 사용한다. 얼굴인식, 차량번호 인식, 개인별 이동경로 저장, 개인 위험점수 부여는 제외한다.

### 6.5 예방 개입 추천

위험 유형별로 안산시가 수행 가능한 조치를 추천한다.

| 위험상황 | 추천 개입 |
|---|---|
| 조위 상승 + 갯벌 방향 이동 증가 | 전광판 문구 변경, 안내방송, 관광안내 채널 복귀 안내 |
| 강풍 + 방파제 체류 증가 | 해안 접근 주의 안내, 순찰 협조 요청 |
| 주말 혼잡 + 일몰 전후 체류 증가 | 사전 알림, 안전요원 배치 권고 |
| 반복 위험구간 발생 | 안내판 위치 개선, 조명 보강, 동선 유도 시설 후보 등록 |
| 관계기관 판단 필요 | 위험 예측 리포트 공유 |

## 8. 개인정보 보호 및 법적 권한 경계

본 시스템은 개인을 식별하거나 추적하지 않는다.

- 얼굴인식 미사용
- 차량번호 인식 미사용
- 개인별 이동경로 저장 금지
- 개인별 위험점수 부여 금지
- 원본 영상 미저장 또는 최소화
- 구역 단위 인원 수, 방향, 체류시간 등 비식별 집계 신호만 활용

또한 본 시스템은 해양경찰의 구조·통제 권한을 대체하지 않는다. 출입통제장소 지정·해제, 강제 통제, 구조 지휘는 관계기관의 법정 권한으로 존중한다. 안산시는 위험 예측, 시민 안내, 시설 개선, 현장 안내 강화, 관계기관 공유 등 지방자치단체가 수행 가능한 예방 행정 조치를 지원한다.

권장 운영 문장:

```text
본 시스템은 해양경찰 등 관계기관의 법정 권한을 대체하지 않는다. 안산시는 공공데이터와 AI를 활용해 위험 시간·장소·상황을 사전에 예측하고, 시민 안내·시설 개선·현장 안내 강화·관계기관 공유 등 지방자치단체가 수행 가능한 예방 행정 조치를 지원한다.
```

## 9. 기대효과

### 시민 안전 향상

위험이 높아지는 시간과 장소를 사전에 파악해 위험 노출을 줄인다.

### 행정 효율 향상

안전 인력, 안내시설, 캠페인, 관계기관 공유를 데이터 기반으로 우선순위화한다.

### 관광 안전 이미지 제고

대부도 등 안산 연안 관광지의 안전 안내 품질을 높여 안전한 관광도시 이미지를 강화한다.

### 정책 개선 근거 확보

반복 위험구간, 안내 효과, 시설 개선 필요성을 데이터로 축적해 예산과 정책 판단 근거로 활용한다.

## 11. 실현가능성 및 추진계획

### 1단계. 공개데이터 기반 MVP

- 조위·기상·사고이력·유동인구 기반 위험지도 구축
- 위험도 산출 및 원인 설명
- 예방 개입 추천 카드 구현
- 관계기관 공유 리포트 템플릿 생성

### 2단계. 시범구간 운영

- 대부도 등 시범구간 후보 선정
- 기존 현장 인프라의 비식별 집계 신호 연계
- 전광판, 관광안내 채널, 현장 안내 프로세스 연동
- 담당자 실행 로그 축적

### 3단계. 효과 검증 및 확대

- 개입 전후 선행지표 비교
- 반복 위험구간 시설 개선 후보 도출
- 안산시 다른 연안·관광 위험공간으로 확대
- 다른 연안 도시, 하천, 호수, 축제장 안전관리 모델로 확산

## 12. 확산 가능성

안산 연안에서 시작한 모델은 다른 공간에도 적용할 수 있다.

- 다른 연안 도시의 갯벌·항구·방파제 안전
- 하천·호수 주변 관광지
- 축제장 혼잡 안전
- 산책로·둘레길 야간 안전
- 폭염·침수 등 생활안전 위험 예측

확산의 핵심은 특정 앱이 아니라 `공공데이터 결합 → 위험 예측 → 예방 개입 추천 → 효과 피드백`이라는 행정 의사결정 구조다.

## 13. 최종 보완 필요사항

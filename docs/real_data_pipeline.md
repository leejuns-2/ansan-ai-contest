# 실제 데이터 수집·모델 학습 파이프라인 v0

## 현재 완료된 실제 데이터 수집

현재 저장소에는 인증키 없이 재현 가능한 실제 데이터 수집 단계가 포함되어 있다.

| 구분 | 상태 | 산출물 |
|---|---|---|
| 해양경찰청 연안사고통계 수집 | 완료 | `data/raw/kcg_coastal_accident_stats_2020_2024.json` |
| 사고유형별 사고이력 prior 모델 학습 | 완료 | `models/history_prior_model.json` |
| 모델 리포트 생성 | 완료 | `models/history_prior_report.md` |
| 대시보드 전처리 데이터 반영 | 완료 | `data/processed/risk_timeseries.json`, `risk_timeseries.js` |

원천:

- 해양경찰청 연안사고통계  
  https://imsm.kcg.go.kr/CSMS/main/csiAcdnt/CsiAcdntSttusRB.do

수집 테이블:

```text
최근 5년('20 ~ '24년) 연안사고 유형별 현황
사고유형: 익수, 추락, 고립, 표류, 기타
지표: 발생 건수, 사망자 수
연도: 2020~2024
```

## 모델 설명

현재 모델은 전체 연안 위험도를 완전히 예측하는 모델이 아니다. 실제 안산 지역 사고 상세 데이터가 확보되기 전 단계에서 사용할 수 있는 `사고이력 prior 모델`이다.

목적:

```text
최근 5년 연안사고 유형별 발생·사망 통계를 이용해
사고유형별 사고이력 prior 점수를 산출한다.
```

산식:

```text
prior_score = clamp(
  50
  + incident_mean_z * 18
  + fatality_rate_z * 16
  + trend_slope_z * 8,
  0,
  100
)
```

현재 장소 유형 매핑:

| 대시보드 후보 구역 | 장소 유형 | 연결한 사고유형 prior |
|---|---|---|
| A01 방아머리 후보 구역 | 갯벌·해안 접근부 | 고립, 익수 |
| A02 탄도항 후보 구역 | 항구·방파제 | 추락, 익수 |
| A03 시화방조제 후보 구역 | 시화호·방조제 인근 | 추락, 고립 |

## 현재 한계

| 한계 | 이유 | 다음 조치 |
|---|---|---|
| 안산 특정 사고 데이터가 아님 | 현재 수집한 KCG 통계는 전국 유형별 집계 | 해양경찰청 연안사고 이력 API/CSV에서 안산·관할 해역 필터 확인 |
| 장소 좌표 기반 모델이 아님 | 수집한 통계에 좌표가 없음 | 연안안전 예측분석모델 격자 데이터 확보 |
| 조위·기상은 샘플 구조 유지 | 공공데이터포털 API 인증키 필요 | `.env`에 인증키를 넣고 API 수집기 확장 |
| 유동인구는 샘플 구조 유지 | 경기데이터드림 원시 데이터 접근 방식 확인 필요 | 다운로드/API 가능 여부 확인 |

## 다음 실제 데이터 수집 대상

### 1. 해양경찰청 연안사고 이력

- 후보: 공공데이터포털 `해양경찰청_연안사고 이력`
- 필요: CSV 다운로드 또는 공공데이터포털 API 인증키
- 목표: 안산·대부도·관할 해경서 단위 필터링
- 활용: 장소 유형, 원인, 요일, 시간대 기반 사고이력 피처

### 2. 국립해양조사원 조위 데이터

- 후보: 조위관측소 실측·예측 조위 조회
- 필요: 공공데이터포털 인증키
- 목표: 안산 연안 대표 관측소 선정
- 활용: 만조까지 남은 시간, 조위 상승 속도

### 3. 기상청 단기예보·초단기실황

- 후보: 기상청 단기예보 조회서비스
- 필요: 공공데이터포털 인증키
- 목표: 대부도·안산 연안 격자 좌표 매핑
- 활용: 풍속, 강수, 기상특보 위험 피처

### 4. 경기데이터드림 유동인구

- 후보: 유동인구 시군구 단위 집계, 방문소비 현황
- 필요: 다운로드/API 가능 여부 확인
- 목표: 방문객 집중 시간대와 평소 대비 혼잡도 산출

## 운영 판단

현재 단계는 `실제 데이터 기반 모델링 착수`로 볼 수 있다. 다만 최종 운영 수준 모델은 아니다. 다음 단계에서 안산 특정 사고이력, 조위, 기상, 유동인구를 모두 수집해 장소-시간 단위 학습 데이터셋을 만들어야 한다.
